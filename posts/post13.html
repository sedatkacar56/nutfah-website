Single vs Double Brackets in R: A Critical Difference

Scenario 1: Reading H5 Files

Read10X_h5(h5_files[1]) returns a list (ERROR)
Read10X_h5(h5_files[[1]]) returns the actual file path string (CORRECT)

Scenario 2: Saving Seurat Objects

seurat_list["sample1"] <- obj stores inside a sublist (INCORRECT)
seurat_list[["sample1"]] <- obj stores as proper named element (CORRECT)

Scenario 3: Accessing Metadata

seurat_list[[1]]@meta.data works correctly
seurat_list[1]@meta.data produces an error

🔑 Key Takeaway

Use [[ ]] when you want the actual object, and [ ] when you want a list of objects. This distinction prevents debugging hours in scRNA-seq pipelines.
