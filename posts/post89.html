The Ellipsis (...) in R — Stop Rewriting Function Arguments

😩TIRED of rewriting the function arguments again and again.🎯💡 Then this post for you!🚀✨

"..." is called ellipsis. It IS A PLACEHOLDER for any number of additional arguments.

- You do not have to rewrite all the arguments in your wrapper function, amazingly "..." will carry them for you. For example:

plot_split <- function(...) {
  plot_clusters(..., col_number = 2, row_number = 2)
}
