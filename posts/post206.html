doyouMind — Cloud Save, Search, and New Features

I use doyouMind to connect my lab, personal life, and ideas in different maps with limitless maps 🧠

New features added based on user feedback:

🚀 Instant Access — Try it instantly — no account needed. Open the app, you're straight in.

⛶ Infinite Canvas — Your ideas have no borders. Drag nodes anywhere, zoom with your scroll wheel.

⬇ Group Drag — Toggle Group Drag ON and dragging any parent node moves its entire branch.

🔍 Search Function — Type a keyword and every matching node lights up yellow.

~~Strikethrough Archive~~ — Apply strikethrough to a node and it fades to 35% opacity.

↩ Undo/Redo with Preview — Before you click Undo, hover the button — it tells you exactly what it will undo.

🎨 Level-based Styling — Every depth level gets its own color and font size automatically.

☁️ Cloud Auto-save + PWA — Sign up and your maps save to the cloud silently as you type.

Try it: doyou-mind.vercel.app. Would love your feedback 🙏
