GitShine — A Free Tool to Beautify Your GitHub Profile

🚀 I built a small open-source web tool — GitShine ✨ Have you ever wanted a beautiful GitHub profile but didn't know where to start? Just enter any GitHub username and GitShine will: 🔍 Fetch your real GitHub data instantly 👤 Show a beautiful profile card with your stats 📊 Visualize your top languages & repositories 📄 Generate a ready-to-use README.md you can copy in one click No login. No sign-up. Completely free.
