tree Command — Simpler Than folder_tree.py

Sometimes you write a whole LinkedIn post, even a GitHub link, just to explain something… and then you learn there was a one-line command all along 😄

I recently shared a workaround to show a folder structure, and Raul Mendez Giraldez, Ph.D. kindly pointed out:

On Linux/macOS/Unix it's simple: just run

tree <path-to-folder>

So I checked it. Yep — it also works in Windows! ✅

And guess what? It works on Windows too (built-in).

Sometimes, to learn a simple tip, we really do need to:
write a full post
make a GitHub repo
and then go back to basics 🙂

Lesson learned:
📌 In the command line, just type tree <path-to-folder>

Back to basics always wins. But still if you want something ready in your current folder jsut to click https://lnkd.in/ggHWiZVg
