Nowadays I'm touching the basics of JavaScript, and whenever you learn a new language, you know, you naturally compare it to the ones you already know. As an R guy, I often catch myself thinking in R's function style or even writing code the way I would in R. Recently, something in JavaScript caught my attention: 👉 You can use object properties without defining them beforehand. Coming from R or other strongly structured languages, this feels unusual at first — but it's one of JavaScript's most powerful features. You can add new fields to an object at any moment, and the engine instantly accepts them. Example let user = { name: "Alice" }; console.log(user); // Output: // { name: "Alice" } user.age = 29; // created dynamically user.isActive = true; // also created dynamically user.settings = { theme: "dark" }; console.log(user); // Output: // { //  name: "Alice", //  age: 29, //  isActive: true, //  settings: { theme: "dark" } // } No declarations. No setup. Just flexible, dynamic objects.
