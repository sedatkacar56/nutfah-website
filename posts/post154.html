folder_tree.py — Extract Single-Cell Project Folder Structure

🔬 Do you need to extract a clean folder tree of your single-cell or spatial transcriptomics project — with all files and subfolders clearly listed (for documentation, sharing, or reproducibility)?

Then it's very simple 👇

Go here https://lnkd.in/gGCNf3pE

Download just two files:
a) folder_tree.py
b) show_folder_tree.bat

Place both files inside your project folder (e.g. a Visium / Space Ranger output, Parse run, or Seurat analysis directory)

Double-click show_folder_tree.bat

Done ✅

📄 The full project structure (raw data, binned outputs, spatial images, analysis files, etc.) is automatically saved as folder_tree_output.txt and opened in Notepad — ready to be shared with collaborators, included in a methods section, or archived for reproducible research. Yea!!! 🚀
