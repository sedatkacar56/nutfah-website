I built a free, open-source application — DualScreen — that stretches HTML content across multiple monitors with synchronized panning and zooming.

No installation required. No coding knowledge. No administrator access. Just download, double-click, select an HTML file, and it works within 10 seconds. The demo shows cars literally driving from monitor 1 to monitor 2.

Version updates:

v1.0.3 — Original HTML-focused release. Portable and installer options available on GitHub.

v1.0.4 — Expanded to support videos, images (jpg, png, gif), and PDFs. Configurable monitor layouts: side-by-side, stacked, 2×2 grid. Fitting options: fill, stretch, crop. Images display as panoramic spans across all screens.

Available on GitHub: github.com/sedatkacar56/dualScreen/releases
