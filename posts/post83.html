Species Gene Nomenclature Mismatch in Human–Rat Single-Cell Integration

PSA: Be careful. Don't try to find shared anchors between human and rat single-cell data (like I did) and then chase this error:

Error in validObject(object = object) : invalid class "Assay5" object: Layers must be two-dimensional objects

I wasted time checking dimensions, cell numbers, Seurat assay conversions, function reproducibility, and package versions… only to realize I was mixing two different gene nomenclatures. 🙃

Lesson: Make sure species/gene symbols match before anchoring (human: PF4, rat: Pf4). Map genes first (e.g., with biomaRt/ortholog tables) or use a species-aware pipeline.
