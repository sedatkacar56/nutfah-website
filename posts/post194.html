doyouMind — A Free Browser-Based Mind Mapping Tool

I built a free browser-based mind mapping tool — doyouMind 🧠 — and have been using it to organize thoughts and brainstorm.

Two key features: 🔹 Auto-building nodes: pressing "+" creates connected child nodes automatically 🔹 Automatic styling: nodes at the same depth share consistent colors and sizes

Other features: 🔹 Drag functionality — moving a node relocates its entire branch 🔹 Infinite canvas — unlimited space, no page constraints 🔹 Mobile support — works on iPhone (via Safari) and Android (via Chrome) as installable web apps 🔹 Voice input — microphone option for creating nodes by speaking

The tool requires email signup with instant access — no confirmation email needed.
