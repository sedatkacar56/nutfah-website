💡 JavaScript Tip: indexOf() Explained Simply Need to check whether a word exists inside a string? That's exactly what indexOf() is for. const text = "JavaScript is awesome!"; text.indexOf("awesome"); // 14 🔹 Returns the position of the substring 🔹 Returns -1 if the substring is not found 🔹 Can start searching from a specific index 🔹 Case-sensitive text.indexOf("JavaScript", 5); If you're checking for existence, remember: if (text.indexOf("awesome") !== -1) { // substring exists} Simple, fast, and still very useful in everyday JavaScript. #JavaScript #WebDevelopment #StringMethods #CleanCode #ProgrammingTips #caseSensitive #AlLatif
